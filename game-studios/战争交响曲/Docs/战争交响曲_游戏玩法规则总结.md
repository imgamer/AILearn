# 《战争交响曲：尼菲林传奇》游戏玩法规则总结

> 本文档基于对原版游戏的深度逆向分析，旨在为换皮项目提供零歧义、可追踪、可直接落地的系统拆解参考。

---

## 一、游戏概览

| 维度 | 内容 |
|------|------|
| 游戏类型 | 回合制战术RPG (Squad-based TRPG) |
| 核心灵感 | 火焰纹章 × 皇家骑士团/欧加战役 |
| 平台 | PC (Steam)，后移植Switch |
| 流程时长 | 30-40小时主线战役，约24章 |
| 画面风格 | 2D像素，Q版战场+真实比例战斗动画 |
| 核心卖点 | 小队编成（组队2小时，战斗5分钟）、50+职业自由转职、士气与投降系统 |
| Steam评分 | 好评如潮 (95%好评，2792篇评测) |

---

## 二、核心循环

```
战役章节(战斗) → Home Base(整备) → 下一章节(战斗) → ...
     ↓                    ↓
  战斗获取资源        招募/转职/科技/羁绊
  经验/CP/物资        装备神器/调整编队
```

**核心心流**：战前编组策略 → 战中战术决策 → 战后养成反馈，三段式循环驱动。

## 三、战斗系统（最核心系统）

### 3.1 小队(Squad)机制

- 每支小队由 **1名队长(Hero) + 最多8名士兵** 组成，上限9人
- 小队容量由队长的 **领导力(LDR)** 决定
- 士兵的 **忠诚度** 影响其占用的容量（忠诚越高，占用越少）
- 小队类型由占多数的单位类型决定，影响移动力和地形适应性

### 3.2 阵型系统（3×5网格）

```
前排(3格): [单位][单位][单位]  ← 承伤/阻挡
中排(3格): [单位][单位][单位]  ← 输出/辅助
后排(3格): [单位][单位][单位]  ← 远程/法师
         + 6个半格位置
```

**关键规则**：
- 前排单位为后排单位 **阻挡近战伤害**
- 一个前排单位可为后排 **4个位置** 提供阻挡
- 重型单位(重步/重骑)可为阻挡范围内的同伴提供 **掩护**，减少伤害
- 多个掩护可 **叠加**，提高伤害减免
- 刺客/盗贼类单位攻击时 **从后排开始**（绕过前排坦克）

### 3.3 攻击顺序（核心策略维度）

不同兵种有不同的攻击轮次优先级：
- **第一轮**：火枪手 → 弓箭手 → 法师（女术士优先）
- **第二轮**：轻步兵 → 剑圣
- **第三轮**：重步兵 → 冠军剑士
- **第四轮**：后续单位

**设计要点**：同一攻击顺序的兵种不要太多，会浪费伤害（集火溢出）。配置多种不同攻击顺序的兵种，实现分批打击，伤害效率最大化。

### 3.4 伤害与防御公式

**伤害类型与护甲交互**：

| 攻击类型 | 对重甲效果 | 备注 |
|---------|-----------|------|
| 劈砍(剑) | 正常伤害 | - |
| 长柄武器 | 正常伤害 | 对骑兵+50%伤害 |
| 打击(锤) | 正常伤害 | 对重步/重骑+25%伤害 |
| 弓箭 | **-50%伤害** | 对重步/重骑严重衰减 |
| 十字弓 | 25%穿甲 | 忽略25%护甲 |
| 火器 | **75%穿甲** | 几乎无视护甲，自身脆弱 |
| 魔法 | **100%穿甲** | 完全忽略重甲 |
| 龙息 | **100%穿甲** | 完全忽略重甲 |

**核心设计哲学**：以高护甲为中心，其他兵种通过各自特性进行有效打击。没有严格的循环克制，而是功能互补。

### 3.5 士气系统（Morale）

| 士气状态 | 效果 |
|---------|------|
| 高士气(我方) | 增加攻击力 + 有几率触发额外行动回合 |
| 低士气(我方) | 显著降低攻击威力 |
| 高士气(敌方) | 显著提升攻击威力 |
| 低士气(敌方) | 降低攻击 + 提升投降几率 |

**士气影响因素**：
- 击败敌人 → 提升
- 与有羁绊关系的英雄共同战斗 → 提升
- 损失人员 → 降低
- 队长倒下 → 大幅降低
- 天气效果 → 根据元素亲和增减

### 3.6 投降系统（Surrender）

- 消灭敌方首领 + 包围敌方小队 → 降低敌人士气 → 增加投降几率
- 威胁等级(Threat Level)高于敌人越多 → 投降几率越高
- 4支部队参与包围时达到最大效果
- 投降的敌人有概率加入我方（成为佣兵）
- **劝降获得的经验值比击杀更高**
- 风险：如果敌人拒绝投降，我方少1次攻击机会且必定后手

### 3.7 地形系统

| 地形 | 效果 |
|------|------|
| 高地(山丘/城墙/堡垒) | 远程射程+1 |
| 森林 | 消耗3移动力，伏击，远程攻击50%闪避 |
| 堡垒/基地 | 防御和闪避加成，被远程攻击时更强防御 |
| 道路 | 移动力上升 |
| 墙壁 | 高地效果，地面无法穿越，可飞越 |
| 村庄/城镇/城堡 | 可占领，获得经验和金币 |
| 矿洞 | 占领获得转职资源(铁/宝石/太阳石等) |
| 神庙 | 复活单位，刷新治疗次数 |
| 集市 | 出售高级佣兵和神器 |

### 3.8 天气系统

| 天气 | 效果 |
|------|------|
| 暴风雪 | 火枪/龙骑兵/炮兵无法开火(除非有后膛装填科技) |
| 热浪(沙漠) | 对无防护单位造成高额士气伤害 |
| 月夜 | 暗/光属性单位增强/削弱 |
| 晴日 | 火/水属性单位增强/削弱 |

天气变化时，对应元素属性的单位会有增强(蓝圈)或削弱(红圈)。

### 3.9 移动类型

| 类型 | 移动力 | 说明 |
|------|--------|------|
| 飞行 | 7 | 忽略地形、城墙、山脉 |
| 轻骑 | 7 | 开阔地形+15%，崎岖地形减弱 |
| 重骑 | 7 | 同轻骑 |
| 重步 | 6 | 标准 |
| 轻步 | 6 | 森林可伏击 |
| 缓慢(法师/炮兵) | 5 | 最慢 |

**混合编队规则**：飞行>慢速>普通>快速。有龙则全队可飞，有慢速单位则全队减速。

**骑兵特殊规则**：攻击后仍可移动（只要移动力未耗尽）。

### 3.10 尼菲林神力（Nephilim Powers）

第18章后主角获得，通过战场菜单使用，消耗神力法力值：

| 神力 | 效果 |
|------|------|
| 祝福(Blessing) | 1支部队额外行动一次，治疗和法术刷新 |
| 范例(Exemplar) | 指定部队本回合+敌方回合50%伤害减免+嘲讽 |
| 传送(Teleport) | 1支部队位移5格内，不消耗行动 |
| 流星(Meteor) | 十字形区域，对敌人造成最多75%最大生命伤害(Boss无效) |
| 恐怖(Terror) | 3×3区域严重降低敌人士气 |

## 四、养成系统

### 4.1 角色属性体系

| 属性 | 缩写 | 影响 |
|------|------|------|
| 力量 | STR | 物理伤害、龙息伤害、物理抗性 |
| 技巧 | SKL | 命中、暴击、闪避、侧击 |
| 魔法 | MAG | 魔法伤害、治疗效果、魔法防御、龙息伤害 |
| 统帅 | LDR | 小队容量(可容纳单位+神器数量) |
| 生命 | HP | 最大生命值 |
| 护甲 | Armor | 物理防御 |

### 4.2 元素亲和系统（6种元素）

| 元素 | 增长倾向 | 适配方向 |
|------|---------|---------|
| 火 | STR++ | 战士系全体 |
| 水 | MAG++ | 法师系、祭司系、女巫系 |
| 电 | SKL++，STR+ | 剑圣系、刺客系 |
| 土 | HP++ | 承伤位(前期拖油瓶，后期核心) |
| 光 | LDR++ | 优秀队长(数量稀少) |
| 暗 | STR+，MAG+ | 力量/魔法双修 |

元素亲和影响：
1. 基础属性的增长速率
2. 天气效果下的命中和暴击几率
3. 可通过"亲和书"修改

### 4.3 经验与职业专精

| 进度条 | 说明 |
|--------|------|
| XP(经验) | 每100升1级，通过战斗/占领/投降/经验书获取 |
| CP(职业专精) | 填满CP条可转职，通过战斗和CP书获取 |

### 4.4 职业转职系统（核心养成深度）

**4阶职业体系，约54种职业**：

#### 一阶(4种基础职业)
| 职业 | 特性 | 定位 |
|------|------|------|
| 勇士(Soldier) | 重甲 | 前排防守 |
| 弓手(Archer) | 远程2格 | 后排输出 |
| 女战士(Valkyrie候选) | 轻装 | 灵活前排 |
| 医疗兵(Medic) | 辅助 | 治疗 |

#### 二阶(15种，新增法术/骑术/长柄3种特性)
- 剑客→剑圣(特化路线)
- 斗士、矛兵、弓骑、骑士、女祭司、学徒等
- 组合特性：如骑术+治疗=疗伤骑手

#### 三阶(25种，新增火器特性)
- 卫兵(重甲+长柄)
- 冠军剑士、百夫长、火枪手、龙骑兵
- 圣骑士、女武神、大法师等
- 任何二阶单位可晋升为火器单位

#### 四阶(龙系+DLC特殊职业)
- 龙骑士、龙法师等

**转职条件**：
1. CP条填满(当前职业专精度满)
2. 属性达标(如转骑兵需20STR+马匹，转法师需10MAG+宝石)
3. 消耗资源(铁锭、宝石、马匹、太阳石等)

**关键设计**：降阶退还资源，调整自由度极高。兵种是"小队的组成材料"，服务于队伍功能搭配。

### 4.5 天赋系统

**单位天赋(生成时自带或通过天赋书获得)**：

核心天赋分类：
- 伤害类：对骑兵+35%、对受伤敌人+25%、对龙+50%
- 防御类：防守-25%伤害、低血-25%伤害、魔法盾50%MAG
- 暴击类：暴击忽略护甲、暴击伤害+50%、暴击回血
- 功能类：击杀回血、防止反击、忽略格挡、幸运提升
- 容量类：±1/2容量占用(与忠诚相关)

**队长天赋(仅当该单位是队长时生效)**：
- 射手/重甲/近战骑兵容量-2
- 劝降几率增加
- 击败后更高物品掉落率
- 防守时护甲+5+经验翻倍
- 免疫热浪/暴雪

### 4.6 羁绊系统（Bond/Relationship）

| 关系类型 | 效果 |
|---------|------|
| 友谊(Friendship) | 共同战斗时士气加成 |
| 羁绊(Bond) | 更强的士气加成 + 永久属性加成 + 专属对话 |
| 家人(Family) | 士气加成 + 剧情变化 |
| 恋爱(Romance) | 最强士气加成 + 结局变化 |

- 主角性别影响恋爱选项，但不限于异性
- 羁绊影响战斗中士气，士气是核心战斗变量
- 羁绊解锁专属对话和剧情分支

## 五、科技树系统

三大分支，4个位阶(T1-T4)，通过阵营评级(Faction Rank)解锁：

### 5.1 左线：战争学院(Arcademy of War)
- **T1**：军事演习(解锁学徒雇佣)★、贸易特使(扩大市场)★★、战场治疗师(辅助HP+20)★
- **T2**：龙之驯养(龙CP+25%)★、军马饲养(骑兵HP+50)★、正义姐妹(重甲魔抗)★、慈悲姐妹(治疗强化)★★
- **T3**：充能魔法(首回合释放)★★、龙骑强化等
- **T4**：终极强化

### 5.2 中线：战术与策略(Tactics & Strategy)
- **T1**：新兵训练(义务兵直接转职)★★、战术加成(15%伤害换35%减伤)★★★
- **T2**：集火队长、劝降强化、勇猛突击
- **T3**：后膛装填(暴雪不影响火器)★★
- **T4**：终极战术

### 5.3 右线：制作与产业(Crafting & Industry)
- **T1**：基础装备制作
- **T2**：高级装备、神器强化
- **T3**：火器生产、攻城器械
- **T4**：终极制造

**关键科技优先级**：中线T1(新兵训练+战术加成) > 左线T1(贸易特使) > 其他

## 六、资源与经济系统

### 6.1 资源类型

| 资源 | 用途 | 获取方式 |
|------|------|---------|
| 金币 | 招募、购买装备/神器 | 战斗、占领、宝箱 |
| 铁锭 | 转职(重甲系) | 占领矿洞 |
| 宝石 | 转职(法师系) | 占领矿洞 |
| 马匹 | 转职(骑兵系) | 占领马厩 |
| 太阳石 | 高级转职(龙系/特殊) | 后期矿洞/商店(稀缺) |
| 亲和书 | 修改元素亲和 | 稀有掉落 |
| 天赋书 | 赋予新天赋 | 稀有掉落 |
| CP书 | 加速职业专精 | 稀有掉落 |

### 6.2 招募系统

| 类型 | 价格 | 特点 |
|------|------|------|
| 义务兵(Conscripts) | 廉价 | 市场前4个栏位，无限量，基础弱 |
| 佣兵(Mercenaries) | 昂贵 | 自带装备和特殊天赋，属性随机 |

**佣兵筛选策略**：存档→批量购买→筛选高初始值→读档→精确购买。关键看力量/灵巧/魔法等核心属性。

### 6.3 神器系统(Artifacts)

- 对整个小队生效的装备
- 效果可正可负
- 科技点满后每队最多装备3件
- 核心神器举例：
  - 时间制控器(防守先手+一轮3次)★★★
  - 罗迪恩之颅(击破招降)
  - 黑带(防守+25%伤害)

## 七、关卡设计

### 7.1 章节结构

- 约24章主线战役
- 每章 = 1场大型战斗
- 章节间 = Home Base整备阶段

### 7.2 胜利条件

| 类型 | 说明 |
|------|------|
| 标准胜利 | 击败Boss + 占领关底大本营 |
| 特殊胜利 | 撤退、护送、坚守等 |

### 7.3 关卡评级

- 占领特殊目标数量
- 击败特殊敌人数量
- 回合数
- 完美评价是后期追求目标

### 7.4 难度设计

| 难度 | 说明 |
|------|------|
| 士兵(Soldier) | 简单，新手入门 |
| 队长(Captain) | 标准 |
| 英雄(Hero) | 困难 |
| 疯狂(Insane) | 解锁条件：通关一次，佣兵永久死亡 |

### 7.5 Home Base(章节间整备)

在Home Base可执行：
- 组织编队(创建/解散/编辑小队)
- 更换职业、装备、特质
- 指派小队长官
- 市场招募(义务兵+佣兵)
- 科技树投资
- 羁绊对话触发
- 竞技场(刷经验和金币)

## 十一、核心设计原则提炼

1. **组队深度 > 战斗操作**：编队策略是核心乐趣，战斗自动结算
2. **以护甲为中心的非对称克制**：不是循环克制，而是功能互补
3. **自由度极高的转职**：降阶退资源，兵种是"材料"而非"身份"
4. **士气是战斗的隐藏变量**：连接羁绊、战术、投降多个系统
5. **渐进式复杂度**：前5章只教基础，中期引入科技/神力，后期才开放龙系
6. **资源稀缺驱动选择**：太阳石/马匹等稀缺资源迫使玩家做取舍
7. **攻击顺序是隐藏的深度策略**：同序兵种集火溢出，异序兵种分批打击
