# S18 属性计算引擎 — 详细需求文档

> 版本：v1.0 | 日期：2026-05-28
> 本文档为零歧义、可追踪、可直接落地的系统需求规格，程序拿到手即可编码。

---

## 一、系统概述

### 1.1 核心定位

统一属性计算引擎，是跨系统的核心基础设施。游戏中11个系统都可能影响最终属性，本引擎定义所有属性加成如何叠加计算，确保数值体系不崩溃。

### 1.2 设计原则

| 原则 | 说明 |
|------|------|
| 统一入口 | 所有系统通过修饰器(MOD)注入，不允许任何系统绕过引擎直接修改属性 |
| 先加后乘 | 同来源的百分比加成先相加，不同来源的百分比加成相乘 |
| 惰性计算 | mod变化时仅标记dirty，下次查询时才重算，避免频繁计算 |
| 可追溯 | 每个mod记录来源系统和来源ID，调试时可完整还原计算过程 |
| 溢出保护 | 每个属性设定上限，乘法叠加后检查上限，HP不可低于1 |

### 1.3 影响属性的11个系统

| 序号 | 系统 | 来源枚举 | 生效范围 | 加成类型 |
|------|------|---------|---------|---------|
| 1 | 基础属性 | Base | 单位级 | 基础值(职业+等级) |
| 2 | 亲和加成 | Affinity | 单位级 | multiply(STR/SKL/MAG/HP) |
| 3 | 天赋加成 | Talent | 单位级 | add + multiply |
| 4 | 装备加成 | Equipment | 单位级 | add |
| 5 | 羁绊加成 | Bond | 单位级 | multiply(全属性) |
| 6 | 科技加成 | Tech | 全局 | multiply(特定属性) |
| 7 | 士气加成 | Morale | 战斗级 | multiply(ATK) |
| 8 | 地形加成 | Terrain | 战斗级 | multiply(AttackRange/DEF) |
| 9 | 天气加成 | Weather | 战斗级 | multiply(Hit/CritRate) |
| 10 | 神器加成 | Artifact | 小队级 | add + multiply |
| 11 | Buff/Debuff | Buff | 战斗级 | add + multiply(有duration) |

### 1.4 需求ID命名规则

| 前缀 | 模块 |
|------|------|
| SE- | 属性计算引擎核心 |
| SM- | 修饰器管理 |

## 三、叠加规则（核心）

### 3.1 叠加规则总览

| 规则 | 说明 | 示例 |
|------|------|------|
| 同来源add叠加 | 同一来源的加法修饰器直接相加 | 天赋A +10ATK + 天赋B +5ATK = +15ATK |
| 同来源multiply先加后乘 | 同一来源的百分比加成先相加，再乘 | 科技A +10%ATK + 科技B +15%ATK = ×1.25 |
| 不同来源multiply相乘 | 不同来源的百分比加成互相相乘 | 天赋×1.2 × 科技×1.25 = ×1.5 |
| Override最高优先 | Override类型直接覆盖最终值 | 某buff使MoveRange固定为0 |

### 3.2 完整计算公式

```
Step1: base_stat = 职业基础值 + 等级成长值(含亲和加成到基础值的部分)

Step2: flat_bonus = SUM(所有add类型mod的value)
       // 包含：天赋add + 装备add + 羁绊add + 神器add + buff_add

Step3: intermediate = base_stat + flat_bonus

Step4: 各来源分别计算multiply总和
       affinity_pct  = SUM(来源=Affinity的multiply mod的value)   // 如0.1+0.05=0.15
       talent_pct    = SUM(来源=Talent的multiply mod的value)
       bond_pct      = SUM(来源=Bond的multiply mod的value)
       tech_pct      = SUM(来源=Tech的multiply mod的value)
       morale_pct    = SUM(来源=Morale的multiply mod的value)
       terrain_pct   = SUM(来源=Terrain的multiply mod的value)
       weather_pct   = SUM(来源=Weather的multiply mod的value)
       artifact_pct  = SUM(来源=Artifact的multiply mod的value)
       buff_pct      = SUM(来源=Buff的multiply mod的value)

Step5: final_stat = intermediate
                   × (1 + affinity_pct)
                   × (1 + talent_pct)
                   × (1 + bond_pct)
                   × (1 + tech_pct)
                   × (1 + morale_pct)
                   × (1 + terrain_pct)
                   × (1 + weather_pct)
                   × (1 + artifact_pct)
                   × (1 + buff_pct)

Step6: final_stat = Floor(final_stat)           // 向下取整
Step7: final_stat = Clamp(final_stat, MIN, MAX) // 限制在[min,max]范围内
```

### 3.3 计算示例

**场景：** 一个战士，基础ATK=100，装备+30ATK，天赋+20%ATK，科技+15%ATK，羁绊+10%ATK，士气+5%ATK

```
Step1: base_stat = 100
Step2: flat_bonus = 30 (装备add)
Step3: intermediate = 100 + 30 = 130
Step4: talent_pct = 0.20, tech_pct = 0.15, bond_pct = 0.10, morale_pct = 0.05
Step5: final_stat = 130 × 1.20 × 1.15 × 1.10 × 1.05
       = 130 × 1.20 = 156
       = 156 × 1.15 = 179.4
       = 179.4 × 1.10 = 197.34
       = 197.34 × 1.05 = 207.207
Step6: Floor(207.207) = 207
Step7: Clamp(207, 1, 9999) = 207
最终ATK = 207
```

### 3.4 关键规则说明

1. **同来源百分比先加后乘**：防止同来源多加成过度叠加。例如3个科技各+20%ATK，是×1.60而非×1.728
2. **不同来源百分比相乘**：鼓励多系统协同。天赋×科技×羁绊的收益高于单一系统堆叠
3. **向下取整**：所有浮点计算完成后统一向下取整，避免浮点精度问题
4. **HP最小为1**：任何情况下HP不低于1，防止除零错误
5. **Override特殊处理**：Override类型的mod在最后阶段生效，直接覆盖计算结果

## 四、完整属性计算流程（C#代码）

### 4.1 StatEngine — 属性计算引擎

```csharp
public class StatEngine
{
    private StatContainer _container;

    public StatEngine(StatContainer container)
    {
        _container = container;
        RebuildIndex();
    }

    public void RegisterMod(StatModifier mod)
    {
        if (mod == null) return;

        // 检查是否已存在同ID的mod
        var existing = _container.modifiers.Find(m => m.mod_id == mod.mod_id);
        if (existing != null)
        {
            // 更新已有mod
            existing.value = mod.value;
            existing.is_active = mod.is_active;
            existing.duration = mod.duration;
            existing.condition_json = mod.condition_json;
        }
        else
        {
            _container.modifiers.Add(mod);
        }

        _container.is_dirty = true;
        RebuildIndex();
    }

    public void RemoveMod(string mod_id)
    {
        int index = _container.modifiers.FindIndex(m => m.mod_id == mod_id);
        if (index >= 0)
        {
            _container.modifiers.RemoveAt(index);
            _container.is_dirty = true;
            RebuildIndex();
        }
    }

    public void RemoveModsBySource(ModSource source, string source_id = null)
    {
        _container.modifiers.RemoveAll(m =>
            m.source_system == source &&
            (source_id == null || m.source_id == source_id));
        _container.is_dirty = true;
        RebuildIndex();
    }

    public void TickDurationMods()
    {
        foreach (var mod in _container.modifiers)
        {
            if (mod.duration > 0)
            {
                mod.duration--;
                if (mod.duration <= 0)
                {
                    mod.is_active = false;
                }
            }
        }
        _container.is_dirty = true;
    }

    public int GetFinalStat(StatType stat_type)
    {
        if (_container.is_dirty)
        {
            Recalculate();
        }
        return _container.cached_final_stats.GetValueOrDefault(stat_type, 0);
    }

    public Dictionary<StatType, int> GetAllFinalStats()
    {
        if (_container.is_dirty)
        {
            Recalculate();
        }
        return new Dictionary<StatType, int>(_container.cached_final_stats);
    }

    public void Recalculate()
    {
        _container.cached_final_stats.Clear();

        foreach (StatType stat_type in Enum.GetValues(typeof(StatType)))
        {
            int final_value = CalculateSingleStat(stat_type);
            _container.cached_final_stats[stat_type] = final_value;
        }

        _container.is_dirty = false;
    }

    private int CalculateSingleStat(StatType stat_type)
    {
        // Step1: 获取基础值
        int base_value = _container.base_stats.GetValueOrDefault(stat_type, 0);

        // 检查Override
        var override_mod = _container.modifiers
            .Where(m => m.stat_type == stat_type && m.mod_type == ModType.Override && m.is_active)
            .OrderBy(m => m.priority)
            .FirstOrDefault();
        if (override_mod != null)
        {
            return ClampStat(stat_type, (int)override_mod.value);
        }

        // Step2: 计算所有add类型mod的总和
        float flat_bonus = _container.modifiers
            .Where(m => m.stat_type == stat_type && m.mod_type == ModType.Add && m.is_active)
            .Sum(m => m.value);

        // Step3: 中间值 = 基础值 + 加法加成
        float intermediate = base_value + flat_bonus;

        // Step4: 按来源分组计算multiply总和
        var multiply_by_source = _container.modifiers
            .Where(m => m.stat_type == stat_type && m.mod_type == ModType.Multiply && m.is_active)
            .GroupBy(m => m.source_system)
            .ToDictionary(g => g.Key, g => g.Sum(m => m.value));

        // Step5: 各来源百分比相乘
        float result = intermediate;
        foreach (var kvp in multiply_by_source)
        {
            result *= (1f + kvp.Value);
        }

        // Step6: 向下取整
        int final_value = Mathf.FloorToInt(result);

        // Step7: 限制范围
        return ClampStat(stat_type, final_value);
    }

    private int ClampStat(StatType stat_type, int value)
    {
        int min = StatCap.MIN_VALUES.GetValueOrDefault(stat_type, 0);
        int max = StatCap.MAX_VALUES.GetValueOrDefault(stat_type, int.MaxValue);
        return Mathf.Clamp(value, min, max);
    }

    private void RebuildIndex()
    {
        _container.modifiers_by_stat = _container.modifiers
            .GroupBy(m => m.stat_type)
            .ToDictionary(g => g.Key, g => g.ToList());

        _container.modifiers_by_source = _container.modifiers
            .GroupBy(m => m.source_system)
            .ToDictionary(g => g.Key, g => g.ToList());
    }

    // 获取属性计算详情(调试用)
    public StatCalculationDetail GetCalculationDetail(StatType stat_type)
    {
        var detail = new StatCalculationDetail
        {
            stat_type = stat_type,
            base_value = _container.base_stats.GetValueOrDefault(stat_type, 0)
        };

        detail.add_mods = _container.modifiers
            .Where(m => m.stat_type == stat_type && m.mod_type == ModType.Add && m.is_active)
            .OrderBy(m => m.priority)
            .ToList();

        detail.flat_bonus = detail.add_mods.Sum(m => m.value);

        detail.multiply_by_source = _container.modifiers
            .Where(m => m.stat_type == stat_type && m.mod_type == ModType.Multiply && m.is_active)
            .GroupBy(m => m.source_system)
            .ToDictionary(
                g => g.Key,
                g => new MultiplyGroup
                {
                    mods = g.OrderBy(m => m.priority).ToList(),
                    total_pct = g.Sum(m => m.value)
                }
            );

        detail.final_value = GetFinalStat(stat_type);
        return detail;
    }
}

public class StatCalculationDetail
{
    public StatType stat_type;
    public int base_value;
    public List<StatModifier> add_mods;
    public float flat_bonus;
    public Dictionary<ModSource, MultiplyGroup> multiply_by_source;
    public int final_value;
}

public class MultiplyGroup
{
    public List<StatModifier> mods;
    public float total_pct;
}
```

## 六、战斗属性 vs 持久属性

### 6.1 属性分类

| 类别 | 计算时机 | 包含的mod来源 | 用途 |
|------|---------|-------------|------|
| 持久属性 | Home Base | Base/Affinity/Talent/Equipment/Bond/Tech/Artifact | 面板显示、编队参考 |
| 战斗属性 | 战斗中 | 持久属性 + Morale/Terrain/Weather/Buff | 实际战斗计算 |

### 6.2 战斗属性计算流程

```
战斗属性 = 持久属性 + 战斗临时mod

具体流程：
1. 进入战斗时，以持久属性为基础
2. 注入战斗临时mod(Morale/Terrain/Weather/Buff)
3. 战斗中查询属性时，引擎自动计算包含临时mod的最终值
4. 战斗结束时，移除所有战斗临时mod
5. 属性恢复为持久属性
```

### 6.3 代码实现

```csharp
public class UnitStatManager
{
    private StatEngine _persistent_engine;  // 持久属性引擎
    private StatEngine _battle_engine;      // 战斗属性引擎(战斗时使用)

    public StatEngine CurrentEngine => _battle_engine ?? _persistent_engine;

    public void EnterBattle()
    {
        // 克隆持久属性容器作为战斗属性基础
        var battle_container = CloneContainer(_persistent_engine.GetContainer());
        _battle_engine = new StatEngine(battle_container);
    }

    public void ExitBattle()
    {
        _battle_engine = null;
    }

    public int GetDisplayStat(StatType stat_type)
    {
        return _persistent_engine.GetFinalStat(stat_type);
    }

    public int GetBattleStat(StatType stat_type)
    {
        return CurrentEngine.GetFinalStat(stat_type);
    }

    private StatContainer CloneContainer(StatContainer source)
    {
        return new StatContainer
        {
            base_stats = new Dictionary<StatType, int>(source.base_stats),
            modifiers = new List<StatModifier>(source.modifiers
                .Where(m => m.source_system != ModSource.Morale
                         && m.source_system != ModSource.Terrain
                         && m.source_system != ModSource.Weather
                         && m.source_system != ModSource.Buff)
                .Select(m => new StatModifier
                {
                    mod_id = m.mod_id,
                    source_system = m.source_system,
                    source_id = m.source_id,
                    stat_type = m.stat_type,
                    mod_type = m.mod_type,
                    value = m.value,
                    priority = m.priority,
                    duration = m.duration,
                    condition_json = m.condition_json,
                    is_active = m.is_active
                })),
            cached_final_stats = new Dictionary<StatType, int>(),
            is_dirty = true
        };
    }
}
```

### 7.1 小队属性公式

### 7.3 代码实现

```csharp
public class SquadStatCalculator
{
    public static SquadStats CalculateSquadStats(Squad squad)
    {
        var stats = new SquadStats();

        foreach (var unit in squad.members)
        {
            float atk_coeff = TB_Class.Get(unit.class_id).atk_coefficient;
            float def_coeff = TB_Class.Get(unit.class_id).def_coefficient;

            stats.total_atk += Mathf.FloorToInt(unit.GetFinalStat(StatType.ATK) * atk_coeff);
            stats.total_def += Mathf.FloorToInt(unit.GetFinalStat(StatType.DEF) * def_coeff);
            stats.total_hp += unit.GetFinalStat(StatType.HP);
        }

        stats.threat_level = CalculateThreatLevel(stats);
        return stats;
    }

    private static int CalculateThreatLevel(SquadStats stats)
    {
        float power = stats.total_atk * 1.0f
                    + stats.total_def * 0.8f
                    + stats.total_hp * 0.1f;
        return Mathf.Max(1, Mathf.FloorToInt(power / 100f));
    }
}

public class SquadStats
{
    public int total_atk;
    public int total_def;
    public int total_hp;
    public int threat_level;
}
```

## 八、数值溢出保护

### 8.1 保护机制

```csharp
public static class OverflowProtection
{
    public static int SafeMultiply(float a, float b)
    {
        float result = a * b;
        if (float.IsInfinity(result) || float.IsNaN(result))
        {
            return StatCap.MAX_VALUES.GetValueOrDefault(StatType.ATK, 9999);
        }
        return Mathf.FloorToInt(result);
    }

    public static int SafeAdd(int a, int b, int max)
    {
        if (a > max - b) return max;
        return a + b;
    }

    public static int ClampToStatRange(StatType stat_type, int value)
    {
        int min = StatCap.MIN_VALUES.GetValueOrDefault(stat_type, 0);
        int max = StatCap.MAX_VALUES.GetValueOrDefault(stat_type, int.MaxValue);
        return Mathf.Clamp(value, min, max);
    }
}
```
