# WarNifeilin 策划文档索引

## 项目简介

基于《战争交响曲：尼菲林传奇》山寨换皮开发的战棋游戏项目。

本文档目录集中存放所有策划相关文档，采用标准化分类和命名规范。

## 📁 目录结构

```
design/
├── 01_requirements/    # 需求文档（核心系统、玩法规则）
├── 02_prototype/       # 原型设计（UI草图、交互原型）
├── 03_flowchart/       # 流程图（系统流程、战斗流程）
├── 04_meeting/         # 会议纪要（需求讨论、评审记录）
├── 05_assets/          # 参考素材（原图、参考截图、参考文档）
├── 06_review/          # 评审记录（版本评审、测试报告）
└── README.md           # 本文档 - 文档索引清单
```
