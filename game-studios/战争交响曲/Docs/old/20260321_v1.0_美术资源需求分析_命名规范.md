## 2. 贴图类型详细说明

### 2.1 漫反射贴图
- **后缀**: `_d` 或 `_diff` 或 `_albedo`
- **用途**: 定义物体表面的基础颜色和纹理
- **格式**: PNG/TGA/PSD
- **色彩空间**: sRGB
- **示例**: `chr_hero01_body_d01.tga`

### 2.2 法线贴图
- **后缀**: `_n` 或 `_nrm` 或 `_normal`
- **用途**: 定义表面凹凸细节，模拟高精度几何
- **格式**: TGA/PNG
- **色彩空间**: Linear
- **通道**: RGB存储法线向量
- **示例**: `chr_hero01_body_n01.tga`

### 2.3 金属度贴图
- **后缀**: `_m` 或 `_metal`
- **用途**: 定义表面金属属性（0=非金属，1=纯金属）
- **格式**: TGA/PNG
- **色彩空间**: Linear
- **通道**: 单通道灰度图
- **示例**: `chr_hero01_armor_m01.tga`

### 2.4 粗糙度贴图
- **后缀**: `_r` 或 `_rough`
- **用途**: 定义表面粗糙度（0=光滑，1=粗糙）
- **格式**: TGA/PNG
- **色彩空间**: Linear
- **通道**: 单通道灰度图
- **示例**: `chr_hero01_armor_r01.tga`

### 2.5 金属度/粗糙度贴图
- **后缀**: `_mr` 或 `_metalrough`
- **用途**: 合并金属度和粗糙度，节省内存
- **格式**: TGA/PNG
- **色彩空间**: Linear
- **通道**: R=金属度，G=粗糙度，B/A=未使用
- **示例**: `chr_hero01_armor_mr01.tga`

### 2.6 自发光贴图
- **后缀**: `_e` 或 `_emissive`
- **用途**: 定义表面自发光区域和强度
- **格式**: TGA/PNG
- **色彩空间**: sRGB
- **通道**: RGB=发光颜色，A=强度
- **示例**: `chr_hero01_weapon_e01.tga`

### 2.7 遮罩贴图
- **后缀**: `_mask` 或 `_msk`
- **用途**: 定义透明区域、混合区域或特殊效果区域
- **格式**: PNG/TGA
- **色彩空间**: Linear
- **通道**: 单通道或多通道灰度图
- **示例**: `env_town_ground_mask01.png`

### 2.8 透明贴图
- **后缀**: `_a` 或 `_alpha` 或 `_trans`
- **用途**: 定义透明度，用于粒子、特效、UI
- **格式**: PNG/TGA
- **色彩空间**: Linear
- **通道**: Alpha通道存储透明度
- **示例**: `fx_skill_fire_a01.png`

### 2.9 立方体贴图
- **后缀**: `_cube` 或 `_cubemap` 或 `_env`
- **用途**: 环境反射、天空盒、光照探针
- **格式**: TGA/HDR/EXR
- **色彩空间**: Linear
- **布局**: 十字布局、水平条带、或6个独立文件
- **示例**: `env_sky_day_cube01.hdr`

### 2.10 高度贴图
- **后缀**: `_h` 或 `_height` 或 `_disp`
- **用途**: 视差贴图或置换贴图
- **格式**: TGA/PNG
- **色彩空间**: Linear
- **通道**: 单通道灰度图
- **示例**: `env_wall_rock_h01.tga`

### 2.11 环境光遮蔽贴图
- **后缀**: `_ao` 或 `_occlusion`
- **用途**: 模拟环境光遮蔽效果
- **格式**: TGA/PNG
- **色彩空间**: Linear
- **通道**: 单通道灰度图
- **示例**: `chr_hero01_body_ao01.tga`

### 2.12 曲率贴图
- **后缀**: `_curv` 或 `_curvature`
- **用途**: 定义表面曲率，用于边缘检测
- **格式**: TGA/PNG
- **色彩空间**: Linear
- **通道**: 单通道灰度图
- **示例**: `chr_hero01_body_curv01.tga`

### 2.13 厚度贴图
- **后缀**: `_thick` 或 `_thickness`
- **用途**: 次表面散射厚度
- **格式**: TGA/PNG
- **色彩空间**: Linear
- **通道**: 单通道灰度图
- **示例**: `chr_skin_face_thick01.tga`

## 3. AI友好命名规范

### 3.1 命名规则格式

#### 标准格式
```
前缀_主体_部位/子对象_用途后缀_通道_版本号.扩展名
```

#### 格式说明
- **前缀**: 资源类型标识（2-3字符小写）
- **主体**: 具体对象名称（字母+数字，下划线分隔）
- **部位/子对象**: 可选，对象的子部分
- **用途后缀**: 可选，描述用途或状态
- **通道**: 贴图类型通道标识
- **版本号**: 2位数字，从01开始
- **扩展名**: 文件格式（小写）

#### 完整示例
```
chr_hero01_body_d01.tga
env_town_market_nrm02.png
ui_icon_bag_spec03.jpg
fx_skill_fire_a01.png
itm_sword_fire_mr01.tga
```

### 3.2 资源类型前缀

| 前缀 | 资源类型 | 说明 |
|------|---------|------|
| chr | Character | 角色资源（英雄、NPC、怪物） |
| env | Environment | 环境资源（场景、地形、建筑） |
| itm | Item | 道具资源（装备、消耗品、材料） |
| ui | UI | 用户界面资源 |
| fx | Effect | 特效资源（粒子、光效） |
| aud | Audio | 音频资源（BGM、音效、语音） |
| tex | Texture | 通用贴图资源 |
| mat | Material | 材质球资源 |
| mesh | Mesh | 网格模型资源 |
| anim | Animation | 动画资源 |
| scn | Scene | 场景文件 |
| prf | Prefab | 预制体资源 |

### 3.3 通道后缀

| 后缀 | 通道类型 | 说明 |
|------|---------|------|
| d / diff / albedo | 漫反射 | 基础颜色贴图 |
| n / nrm / normal | 法线 | 法线贴图 |
| m / metal | 金属度 | 金属度贴图 |
| r / rough | 粗糙度 | 粗糙度贴图 |
| mr / metalrough | 金属度/粗糙度 | 合并贴图 |
| e / emissive | 自发光 | 自发光贴图 |
| a / alpha / trans | 透明 | 透明贴图 |
| mask / msk | 遮罩 | 遮罩贴图 |
| cube / cubemap / env | 立方体 | 立方体贴图 |
| h / height / disp | 高度 | 高度/置换贴图 |
| ao / occlusion | 环境光遮蔽 | AO贴图 |
| curv / curvature | 曲率 | 曲率贴图 |
| thick / thickness | 厚度 | 厚度贴图 |
| spec | 高光 | 高光贴图（旧版） |

### 3.4 版本号规则

#### 格式
- 2位数字：`01` - `99`
- 递增规则：每次修改+1
- 重置规则：重大修改重置为01

#### 示例
```
chr_hero01_body_d01.tga  # 初始版本
chr_hero01_body_d02.tga  # 第一次修改
chr_hero01_body_d03.tga  # 第二次修改
```

### 3.5 扩展名规范

| 扩展名 | 用途 | 说明 |
|--------|------|------|
| .tga | 贴图 | 通用贴图格式，支持Alpha通道 |
| .png | 贴图/UI | 无损压缩，适合UI和透明贴图 |
| .jpg | 贴图 | 有损压缩，适合大尺寸漫反射贴图 |
| .psd | 源文件 | Photoshop源文件，保留图层 |
| .fbx | 模型 | 3D模型格式，支持动画 |
| .obj | 模型 | 3D模型格式，仅几何体 |
| .unity | 场景 | Unity场景文件 |
| .prefab | 预制体 | Unity预制体 |
| .mat | 材质 | Unity材质球 |
| .anim | 动画 | Unity动画片段 |
| .controller | 动画控制器 | Unity动画控制器 |

## 5. Python批处理脚本框架

### 5.1 资产校验脚本

```python
import os
import re
import csv
from pathlib import Path
from typing import Dict, List, Tuple

class AssetValidator:
    def __init__(self, root_path: str):
        self.root_path = Path(root_path)
        self.valid_extensions = {'.tga', '.png', '.jpg', '.psd', '.fbx', '.obj', 
                                 '.unity', '.prefab', '.mat', '.anim', '.controller',
                                 '.wav', '.mp3', '.ogg', '.json', '.xml'}
        self.prefixes = {'chr', 'env', 'itm', 'ui', 'fx', 'aud', 'tex', 'mat', 
                         'mesh', 'anim', 'scn', 'prf'}
        self.channels = {'d', 'diff', 'albedo', 'n', 'nrm', 'normal', 'm', 'metal',
                         'r', 'rough', 'mr', 'metalrough', 'e', 'emissive', 'a',
                         'alpha', 'trans', 'mask', 'msk', 'cube', 'cubemap', 'env',
                         'h', 'height', 'disp', 'ao', 'occlusion', 'curv', 
                         'curvature', 'thick', 'thickness', 'spec'}
        
        # 完整文件名正则
        self.filename_pattern = re.compile(
            r'^(?P<prefix>[a-z]{2,3})_'
            r'(?P<subject>[a-z0-9]+)'
            r'(?:_(?P<part>[a-z0-9]+))?'
            r'(?:_(?P<purpose>[a-z0-9]+))?'
            r'_(?P<channel>[a-z]+)'
            r'(?P<version>[0-9]{2})'
            r'\.(?P<ext>[a-z]{3,4})$'
        )
    
    def validate_filename(self, filename: str) -> Tuple[bool, str, Dict]:
        """
        验证文件名是否符合命名规范
        
        Args:
            filename: 文件名
            
        Returns:
            (是否有效, 错误信息, 解析结果字典)
        """
        # 检查长度
        if len(filename) > 64:
            return False, f"文件名过长: {len(filename)} > 64", {}
        
        # 检查扩展名
        ext = Path(filename).suffix.lower()
        if ext not in self.valid_extensions:
            return False, f"无效的扩展名: {ext}", {}
        
        # 检查命名规范
        match = self.filename_pattern.match(filename)
        if not match:
            return False, "文件名不符合命名规范", {}
        
        result = match.groupdict()
        
        # 检查前缀
        if result['prefix'] not in self.prefixes:
            return False, f"无效的前缀: {result['prefix']}", result
        
        # 检查通道
        channel = result['channel']
        # 提取通道部分（去掉版本号）
        channel_base = ''.join(filter(str.isalpha, channel))
        if channel_base not in self.channels:
            return False, f"无效的通道: {channel_base}", result
        
        return True, "文件名有效", result
    
    def validate_directory(self, directory: Path) -> List[Dict]:
        """
        验证目录下所有文件
        
        Args:
            directory: 目录路径
            
        Returns:
            验证结果列表
        """
        results = []
        
        for file_path in directory.rglob('*'):
            if file_path.is_file():
                filename = file_path.name
                is_valid, message, parsed = self.validate_filename(filename)
                
                results.append({
                    'path': str(file_path.relative_to(self.root_path)),
                    'filename': filename,
                    'is_valid': is_valid,
                    'message': message,
                    'parsed': parsed
                })
        
        return results
    
    def generate_report(self, results: List[Dict], output_path: str):
        """
        生成验证报告CSV
        
        Args:
            results: 验证结果列表
            output_path: 输出CSV路径
        """
        with open(output_path, 'w', newline='', encoding='utf-8') as csvfile:
            fieldnames = ['path', 'filename', 'is_valid', 'message', 
                         'prefix', 'subject', 'part', 'purpose', 
                         'channel', 'version', 'ext']
            writer = csv.DictWriter(csvfile, fieldnames=fieldnames)
            writer.writeheader()
            
            for result in results:
                row = {
                    'path': result['path'],
                    'filename': result['filename'],
                    'is_valid': result['is_valid'],
                    'message': result['message']
                }
                
                # 添加解析结果
                if result['parsed']:
                    row.update(result['parsed'])
                
                writer.writerow(row)
        
        print(f"验证报告已生成: {output_path}")
    
    def print_summary(self, results: List[Dict]):
        """
        打印验证摘要
        
        Args:
            results: 验证结果列表
        """
        total = len(results)
        valid = sum(1 for r in results if r['is_valid'])
        invalid = total - valid
        
        print(f"\n验证摘要:")
        print(f"  总文件数: {total}")
        print(f"  有效文件: {valid}")
        print(f"  无效文件: {invalid}")
        
        if invalid > 0:
            print(f"\n无效文件列表:")
            for result in results:
                if not result['is_valid']:
                    print(f"  - {result['path']}: {result['message']}")


# 使用示例
if __name__ == "__main__":
    validator = AssetValidator("Assets/Art")
    results = validator.validate_directory(Path("Assets/Art"))
    validator.generate_report(results, "asset_validation_report.csv")
    validator.print_summary(results)
```

### 5.2 批量重命名脚本

```python
import os
import re
from pathlib import Path
from typing import Dict, Optional

class AssetRenamer:
    def __init__(self, root_path: str, dry_run: bool = True):
        self.root_path = Path(root_path)
        self.dry_run = dry_run  # 干运行模式，不实际重命名
        
        # 旧文件名到新文件名的映射
        self.rename_rules = {
            # 示例规则
            r'Hero01_Body_D01\.tga': 'chr_hero01_body_d01.tga',
            r'Hero01_Body_N01\.tga': 'chr_hero01_body_n01.tga',
            r'Town_Market_Nrm02\.png': 'env_town_market_nrm02.png',
        }
    
    def normalize_filename(self, filename: str) -> str:
        """
        标准化文件名
        
        Args:
            filename: 原文件名
            
        Returns:
            标准化后的文件名
        """
        # 转换为小写
        filename = filename.lower()
        # 替换空格和连字符为下划线
        filename = re.sub(r'[ -]', '_', filename)
        # 移除特殊字符（保留字母、数字、下划线、点）
        filename = re.sub(r'[^a-z0-9_.]', '', filename)
        return filename
    
    def apply_rename_rules(self, filename: str) -> Optional[str]:
        """
        应用重命名规则
        
        Args:
            filename: 原文件名
            
        Returns:
            新文件名，如果没有匹配规则则返回None
        """
        for pattern, new_name in self.rename_rules.items():
            if re.match(pattern, filename, re.IGNORECASE):
                return new_name
        return None
    
    def rename_file(self, file_path: Path) -> bool:
        """
        重命名单个文件
        
        Args:
            file_path: 文件路径
            
        Returns:
            是否成功重命名
        """
        old_name = file_path.name
        
        # 先尝试应用重命名规则
        new_name = self.apply_rename_rules(old_name)
        
        # 如果没有匹配规则，则标准化文件名
        if not new_name:
            new_name = self.normalize_filename(old_name)
        
        # 如果文件名没有变化，跳过
        if new_name == old_name:
            return False
        
        new_path = file_path.parent / new_name
        
        # 检查新文件名是否已存在
        if new_path.exists():
            print(f"警告: 目标文件已存在，跳过: {new_path}")
            return False
        
        # 执行重命名
        if self.dry_run:
            print(f"[DRY RUN] {file_path} -> {new_path}")
            return True
        else:
            try:
                file_path.rename(new_path)
                print(f"重命名: {file_path} -> {new_path}")
                return True
            except Exception as e:
                print(f"错误: 重命名失败 {file_path}: {e}")
                return False
    
    def rename_directory(self, directory: Path) -> Dict:
        """
        重命名目录下所有文件
        
        Args:
            directory: 目录路径
            
        Returns:
            重命名统计信息
        """
        stats = {
            'total': 0,
            'renamed': 0,
            'skipped': 0,
            'failed': 0
        }
        
        for file_path in directory.rglob('*'):
            if file_path.is_file():
                stats['total'] += 1
                
                if self.rename_file(file_path):
                    stats['renamed'] += 1
                else:
                    stats['skipped'] += 1
        
        return stats
    
    def print_summary(self, stats: Dict):
        """
        打印重命名摘要
```

### 5.3 资产清单生成脚本

```python
import os
import csv
import json
from pathlib import Path
from typing import Dict, List, Any
from collections import defaultdict

class AssetInventoryGenerator:
    def __init__(self, root_path: str):
        self.root_path = Path(root_path)
        
        # 文件类型分类
        self.texture_extensions = {'.tga', '.png', '.jpg', '.psd', '.hdr', '.exr'}
        self.model_extensions = {'.fbx', '.obj', '.max', '.ma', '.mb'}
        self.animation_extensions = {'.anim', '.controller'}
        self.audio_extensions = {'.wav', '.mp3', '.ogg'}
        self.data_extensions = {'.json', '.xml', '.csv'}
        
        # 资源类型映射
        self.resource_types = {
            'chr': 'Character',
            'env': 'Environment',
            'itm': 'Item',
            'ui': 'UI',
            'fx': 'Effect',
            'aud': 'Audio',
            'tex': 'Texture',
            'mat': 'Material',
            'mesh': 'Mesh',
            'anim': 'Animation',
            'scn': 'Scene',
            'prf': 'Prefab'
        }
    
    def get_file_info(self, file_path: Path) -> Dict[str, Any]:
        """
        获取文件信息
        
        Args:
            file_path: 文件路径
            
        Returns:
            文件信息字典
        """
        stat = file_path.stat()
        filename = file_path.name
        ext = file_path.suffix.lower()
        
        # 解析文件名
        parts = filename.split('_')
        prefix = parts[0] if len(parts) > 0 else ''
        resource_type = self.resource_types.get(prefix, 'Unknown')
        
        # 确定文件类别
        if ext in self.texture_extensions:
            file_category = 'Texture'
        elif ext in self.model_extensions:
            file_category = 'Model'
        elif ext in self.animation_extensions:
            file_category = 'Animation'
        elif ext in self.audio_extensions:
            file_category = 'Audio'
        elif ext in self.data_extensions:
            file_category = 'Data'
        else:
            file_category = 'Other'
        
        return {
            'path': str(file_path.relative_to(self.root_path)),
            'filename': filename,
            'extension': ext,
            'size_bytes': stat.st_size,
            'size_mb': round(stat.st_size / (1024 * 1024), 2),
            'prefix': prefix,
            'resource_type': resource_type,
            'file_category': file_category,
            'subject': parts[1] if len(parts) > 1 else '',
        }
    
    def scan_directory(self, directory: Path) -> List[Dict]:
        """
        扫描目录下所有文件
        
        Args:
            directory: 目录路径
            
        Returns:
            文件信息列表
        """
        files = []
        
        for file_path in directory.rglob('*'):
            if file_path.is_file():
                files.append(self.get_file_info(file_path))
        
        return files
    
    def generate_inventory(self, files: List[Dict]) -> Dict[str, Any]:
        """
        生成资产清单
        
        Args:
            files: 文件信息列表
            
        Returns:
            资产清单字典
        """
        inventory = {
            'total_files': len(files),
            'total_size_mb': round(sum(f['size_mb'] for f in files), 2),
            'by_resource_type': defaultdict(lambda: {'count': 0, 'size_mb': 0}),
            'by_file_category': defaultdict(lambda: {'count': 0, 'size_mb': 0}),
            'by_extension': defaultdict(lambda: {'count': 0, 'size_mb': 0}),
            'files': files
        }
        
        for file_info in files:
            # 按资源类型统计
            rt = file_info['resource_type']
            inventory['by_resource_type'][rt]['count'] += 1
            inventory['by_resource_type'][rt]['size_mb'] += file_info['size_mb']
            
            # 按文件类别统计
            fc = file_info['file_category']
            inventory['by_file_category'][fc]['count'] += 1
            inventory['by_file_category'][fc]['size_mb'] += file_info['size_mb']
            
            # 按扩展名统计
            ext = file_info['extension']
            inventory['by_extension'][ext]['count'] += 1
            inventory['by_extension'][ext]['size_mb'] += file_info['size_mb']
        
        return inventory
    
    def save_csv(self, inventory: Dict[str, Any], output_path: str):
        """
        保存CSV格式清单
        
        Args:
            inventory: 资产清单
            output_path: 输出路径
        """
        with open(output_path, 'w', newline='', encoding='utf-8') as csvfile:
            fieldnames = ['path', 'filename', 'extension', 'size_bytes', 
                         'size_mb', 'prefix', 'resource_type', 
                         'file_category', 'subject']
            writer = csv.DictWriter(csvfile, fieldnames=fieldnames)
            writer.writeheader()
            
            for file_info in inventory['files']:
                writer.writerow(file_info)
        
        print(f"CSV清单已生成: {output_path}")
    
    def save_json(self, inventory: Dict[str, Any], output_path: str):
        """
        保存JSON格式清单
        
        Args:
            inventory: 资产清单
            output_path: 输出路径
        """
        # 转换defaultdict为普通dict
        inventory_copy = dict(inventory)
        inventory_copy['by_resource_type'] = dict(inventory['by_resource_type'])
        inventory_copy['by_file_category'] = dict(inventory['by_file_category'])
        inventory_copy['by_extension'] = dict(inventory['by_extension'])
        
        with open(output_path, 'w', encoding='utf-8') as jsonfile:
            json.dump(inventory_copy, jsonfile, indent=2, ensure_ascii=False)
        
        print(f"JSON清单已生成: {output_path}")
    
    def print_summary(self, inventory: Dict[str, Any]):
        """
        打印清单摘要
        
        Args:
            inventory: 资产清单
        """
        print(f"\n资产清单摘要:")
        print(f"  总文件数: {inventory['total_files']}")
        print(f"  总大小: {inventory['total_size_mb']} MB")
        
        print(f"\n按资源类型统计:")
        for rt, stats in sorted(inventory['by_resource_type'].items()):
            print(f"  {rt}: {stats['count']} 文件, {stats['size_mb']:.2f} MB")
        
        print(f"\n按文件类别统计:")
        for fc, stats in sorted(inventory['by_file_category'].items()):
            print(f"  {fc}: {stats['count']} 文件, {stats['size_mb']:.2f} MB")
        
        print(f"\n按扩展名统计:")
        for ext, stats in sorted(inventory['by_extension'].items()):
            print(f"  {ext}: {stats['count']} 文件, {stats['size_mb']:.2f} MB")


# 使用示例
if __name__ == "__main__":
    generator = AssetInventoryGenerator("Assets")
    files = generator.scan_directory(Path("Assets"))
    inventory = generator.generate_inventory(files)
    
    # 生成报告
    generator.save_csv(inventory, "asset_inventory.csv")
    generator.save_json(inventory, "asset_inventory.json")
    generator.print_summary(inventory)
```
