## 1. 玩家分群画像与核心动机

| 玩家类型 | 画像描述 | 核心动机 | 占比预估 |
|----------|----------|----------|----------|
| **策略爱好者** | 喜欢战棋，喜欢思考阵容搭配，享受战略决策 | 通过智力和策略战胜强敌，体验布阵的乐趣 | 30% |
| **收集养成爱好者** | 喜欢收集英雄，培养后代，打造完美队伍 | 收集全角色，养成满属性极品角色，完成图鉴 | 25% |
| **剧情爱好者** | 喜欢看故事，体验关卡剧情，跟着流程走 | 享受剧情推进，完成整个游戏故事体验 | 20% |
| **挑战爱好者** | 喜欢高难度挑战，追求完美通关 | 挑战最高难度，完成全成就，证明自己实力 | 15% |
| **休闲玩家** | 碎片化玩，不想太费脑，喜欢自动战斗 | 轻松体验，不怎么操作就能体验游戏内容 | 10% |

## 2. 用户故事地图总览

```
玩家旅程：
  新手入门 → 阵容组建 → 科技研发 → 羁绊养成 → 婚姻生育 → 关卡推进
    ↓
  开放探索 → 装备收集 → 竞技场挑战 → 神器打造 → 阵营升级
    ↓
  终局BOSS → 成就解锁 → New Game+ → 多周目挑战
```

## 3. 第一章：新手引导阶段

### 3.1 首次进入游戏

| 项目 | 内容 |
|------|------|
| **用户故事** | 作为**新玩家**，我想要**顺利进入游戏并看到清晰的主菜单**，从而**快速开始游戏，不会迷路** |
| **关联系统** | 主界面系统 |
| **玩法规则** | - 点击"新游戏"开始新存档<br>- 点击"继续游戏"读取最近存档<br>- 点击"设置"调整音量难度 |

| **测试用例** | 1. 首次进入能正常显示菜单 ✓<br>2. 点击新游戏能正常进入新手关卡 ✓<br>3. 无存档时继续游戏不可点 ✓ |

| **Should Have** | 应该有 | 重要功能，尽量在当前版本做，实在不行可以延后 |
