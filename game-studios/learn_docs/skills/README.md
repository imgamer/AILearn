## 🎯 学习目标

完成本学习路径后，你将能够：

- ✅ 理解 Skill 系统的核心概念和设计哲学
- ✅ 独立创建符合最佳实践的 Skill
- ✅ 选择合适的 Skill 类型解决特定问题
- ✅ 调试和优化 Skill 性能
- ✅ 设计复杂的多步骤工作流程
- ✅ 为团队贡献可复用的 Skill

## 🚀 快速开始

### 方式一：系统学习（推荐）

按顺序阅读 [01-what-is-skill.md](./01-what-is-skill.md) 开始，
跟随每个文档的动手练习。

### 方式二：问题导向

| 我想... | 阅读文档 |
|---------|----------|
| 快速理解 Skill 是什么 | [01-what-is-skill.md](./01-what-is-skill.md) |
| 立即创建一个 Skill | [04-your-first-skill.md](./04-your-first-skill.md) |
| 理解不同 Skill 类型的区别 | [03-skill-types.md](./03-skill-types.md) |
| 解决一个具体问题的最佳实践 | [09-case-studies.md](./09-case-studies.md) |
| 查找语法快速参考 | [10-quick-reference.md](./10-quick-reference.md) |

### 方式三：通过示例学习

直接进入 [examples/](./examples/) 目录：
1. 先阅读 `README.md` 了解示例概览
2. 选择一个与你需求最接近的示例
3. 通读代码和注释
4. 复制并修改为你自己的 Skill
