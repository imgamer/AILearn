# BlueprintNodeBridge 用户手册

## 简介

BlueprintNodeBridge 是一款为 Unreal Engine 5.7 蓝图编辑器开发的插件工具，提供节点复制粘贴、布局整理和 AI 规则管理功能。

## 功能概述

### AI Bridge 菜单

插件在以下编辑器工具栏添加 **AI Bridge** 下拉菜单：

| 菜单项 | 功能说明 |
|--------|----------|
| 粘贴短代码 | 从剪贴板读取短代码并粘贴节点 |
| 复制为短代码 | 将选中节点转换为短代码并复制到剪贴板 |
| 整理节点 | 自动排列选中的多个节点 |
| 复制 AI 规则 | 复制 AI 生成规则说明到剪贴板 |
| 管理模板 | 打开节点模板缓存文件夹 |
| 编辑 AI 规则 | 用外部编辑器打开 AI 规则 JSON 文件 |

## 使用方法

### 1. 复制蓝图节点为短代码

1. 在蓝图编辑器中选中一个或多个节点
2. 点击工具栏 **AI Bridge** → **复制为短代码**
3. 短代码自动复制到剪贴板
4. 系统自动保存节点模板到缓存

### 2. 粘贴短代码

1. 复制短代码文本（格式为 `NodeName_ID (...) : (...);`）
2. 在蓝图编辑器中点击工具栏 **AI Bridge** → **粘贴短代码**
3. 节点将粘贴到当前视图中心位置

### 3. 整理节点布局

1. 选中 2 个或更多节点
2. 点击 **AI Bridge** → **整理节点**
3. 节点按执行流程自动排列

### 4. 管理 AI 规则

- **复制 AI 规则**: 获取短代码格式规范，用于 AI 生成
- **编辑 AI 规则**: 自定义 `AIRules.json` 文件
- **管理模板**: 手动编辑 `NodeTemplates.json` 缓存文件

## 短代码格式说明

短代码采用紧凑格式表示蓝图节点：

```
NodeName_UniqueID (InputPins;) : (OutputPins;)
```

**示例：**
```
Branch_1 (inExec; Condition\(true\);) : (outExec True; outExec False;)
PrintString_2 (inExec; InString\("Hello"\);) : (outExec;)
```

**特殊字符转义：**
- `(` → `\(` 
- `)` → `\)`
- `;` → `\;`
- `\` → `\\`
- 换行 → `\n`

## 常见问题

**Q: 粘贴失败怎么办？**
- 确认剪贴板内容是有效的短代码格式
- 确保当前焦点在蓝图编辑器窗口

**Q: 整理节点不工作？**
- 至少需要选中 2 个节点
- 检查节点间是否有执行连接线

**Q: 如何查看已缓存的模板？**
- 使用"管理模板"功能打开缓存文件夹
