版本对比
  - Python 3.8/3.9：已 EOL，不推荐
  - Python 3.11：支持至 2027-10，难度中等，性能提升显著 ← 推荐
  - Python 3.12：难度高（废弃 API 正式移除）
  - Python 3.13：难度极高（大量 API 变更 + 实验性功能）
  三阶段升级路线
  1. Phase 0 (1-2天)：代码审计 + 测试基准
  2. Phase 1 (1周)：Python 3.8 过渡验证（可选跳过）
  3. Phase 2 (2-3周)：Python 3.11 目标迁移 + 全面测试
