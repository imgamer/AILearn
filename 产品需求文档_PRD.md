## 一、产品定位

### 1.1 一句话描述
小队编成为核心乐趣的回合制战术RPG手游，玩家组建最多9人小队，利用阵型、职业搭配、士气与地形在大型战场上智取敌人。

### 1.2 目标用户画像

| 画像 | 特征 |
|------|------|
| 核心用户 | 25-40岁，有火纹/梦幻模拟战/皇家骑士团经验的硬核战棋玩家 |
| 次核心用户 | 20-35岁，卡牌养成+SLG玩家，喜欢"组队2小时"的深度搭配 |
| 泛用户 | 18-30岁，被美术/剧情吸引，可接受自动战斗的轻度策略玩家 |

### 1.3 核心差异化

| 维度 | 市场现状 | 本项目 |
|------|---------|--------|
| 战斗单位 | 1角色=1单位 | **1小队=最多9单位**，阵型+编队深度 |
| 克制关系 | 严格循环克制 | **以护甲为中心的功能互补**，非对称克制 |
| 转职 | 线性升级 | **网状自由转职**，降阶退资源 |
| 士气 | 无/纯数值 | **士气→再动/投降**，连接羁绊与战术 |
| 战场规模 | 小地图6v6 | **大地图多小队**，包围/占领/地形 |

### 5.2 难度曲线设计

**关卡难度曲线(核心数值建模)**：

```
章节:  1    5    10   15   20   25   30
       │    │    │    │    │    │    │
敌HP:  100  250  500  900  1400 2000 2800
敌ATK: 15   35   65   100  145  195  255
敌DEF: 5    15   30   50   75   105  140
我方等级:1    8    16   24   32   40   50
```

**难度曲线公式**：
- 敌方HP = 基础HP × (1 + 章节数 × 0.12) ^ 2
- 敌方ATK = 基础ATK × (1 + 章节数 × 0.10) ^ 1.8
- 敌方DEF = 基础DEF × (1 + 章节数 × 0.08) ^ 1.6
- 我方预期等级 = 章节数 × 1.6 + 1

**难度跳跃点(卡点设计)**：
- 第5章：首次出现骑兵敌人 → 考验长柄兵种搭配
- 第10章：首次出现重甲敌人 → 考验穿甲/魔法配置
- 第15章：首次出现龙系敌人 → 考验对空/高伤配置
- 第20章：Boss战，多波次 → 考验续航和资源管理
- 第25章：敌方有神力 → 考验反制策略
- 第30章：最终Boss → 综合考验

## 七、数值建模核心公式

### 7.1 伤害公式

```
基础伤害 = 攻击方ATK × 技能倍率 - 防御方DEF × (1 - 穿甲率)

实际伤害 = 基础伤害 × (1 + 士气加成) × (1 + 地形加成) × (1 + 天气加成)
         × (1 + 掩护减免) × (1 + 羁绊加成) × (1 + 科技加成)
         × 暴击系数(1.5/2.0) × 随机浮动(0.9~1.1)
```

### 7.2 属性成长公式

```
每级属性增长 = 基础成长率 × (1 + 亲和加成% + 天赋加成% + 装备加成% + 羁绊加成%)

属性成长率参考表(每级)：
         STR   SKL   MAG   LDR   HP
火亲和:  +2.5  +1.0  +0.5  +0.5  +8
水亲和:  +0.5  +1.0  +2.5  +0.5  +8
电亲和:  +1.5  +2.5  +0.5  +0.5  +6
土亲和:  +1.0  +0.5  +0.5  +0.5  +12
光亲和:  +1.0  +0.5  +1.0  +2.5  +8
暗亲和:  +2.0  +0.5  +2.0  +0.5  +6
```

### 7.3 容量公式

```
小队容量 = 队长LDR × 2 + 队长等级 × 0.5

单位占用 = 基础占用(8) - 忠诚加成(0~3) - 天赋减免(0~2) - 队长天赋减免(0~2)
神器占用 = 品质占用(白1/绿2/蓝3/紫4/橙5)
```

### 7.4 投降概率公式

```
投降概率 = MIN(80%, MAX(0%,
  (我方威胁等级 - 敌方威胁等级) / 敌方威胁等级 × 50%
  + (50 - 敌方士气) / 50 × 30%
  + 包围加成(0/5/10/15)%
  + 科技加成%
  + 队长天赋加成%
))
```

### 7.5 威胁等级公式

```
威胁等级 = SUM(单位等级 × 职业系数 × 品质系数) + 神器总价值/100

职业系数: 一阶1.0 / 二阶1.5 / 三阶2.2 / 四阶3.5
品质系数: N1.0 / R1.1 / SR1.3 / SSR1.6
```
