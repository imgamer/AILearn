CVAT (Computer Vision Annotation Tool) 是一个用于图像和视频标注的开源工具，旨在帮助用户完成计算机视觉项目中的数据标注工作。
该网站提供了 CVAT 在线版本，用户无需下载安装即可直接在线使用。
支持多种标注类型，如边界框、多边形、关键点等，适用于对象检测、图像分割、动作识别等多种任务。
提供协作功能，允许多个标注员同时参与同一个项目，适合团队合作。
具备数据管理功能，方便用户上传、存储和管理标注数据。
可以与机器学习模型集成，支持预标注和模型训练后的标注验证，加速数据准备过程。

以下是使用 LABELME 进行图片标注的完整过程： 
2.1、安装 LABELME 
2.1.1. 打开命令提示符或终端窗口。
2.1.2. 使用以下命令安装 LABELME（如果使用 Anaconda 环境，可以在 Anaconda Prompt 中执行）：
 pip install labelme  
2.2、启动 LABELME 
在命令提示符或终端中输入：
 labelme 
这将启动 LABELME 应用程序。 
2.3、加载图片 
在 LABELME 界面中，点击“Open Dir”选择包含要标注图片的文件夹，或者点击“Open”直接选择一张图片进行标注。 
2.4、标注操作 
2.4.1. 使用左侧的标注工具进行标注：
(1)	“多边形”工具：用于绘制不规则形状的标注区域，依次点击多个点形成多边形。
(2)	“矩形”工具：绘制矩形标注区域。
(3)	“点”工具：标注单个点。
2.4.2. 对于每个标注区域，可以在右侧的“标签列表”中选择或添加合适的标签名称来标识该区域的类别。 
2.5、保存标注 
1. 标注完成后，点击“Save”按钮保存标注结果。
2. LABELME 会将标注结果保存为 JSON 文件，文件名与对应的图片文件名相同，保存在与图片相同的文件夹中。 
2.6、查看和编辑标注 
1. 可以随时再次打开已标注的图片和 JSON 文件进行查看和编辑。
2. 对标注进行修改、添加或删除操作，然后再次保存。
